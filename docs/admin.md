## Краткая сводка команд

| Раздел | Ключевые команды |
| --- | --- |
| Диски | `df -h`, `du -sh`, `smartctl` |
| Мониторинг | `top`, `htop`, `iostat`, `iotop` |
| Файлы | `ls`, `cp`, `mv`, `rm`, `cat`, `grep` |
| Сеть | `ping`, `traceroute`, `nmap`, `netstat` |
| Пользователи | `useradd`, `usermod`, `visudo` |
