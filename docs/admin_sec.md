# Безопасная разработка (DevSecOps)

## Технические меры обеспечения безопасности разработки

!!! info "Технические меры безопасности"
    | Мера | Описание |
    |------|----------|
    | **ASPM** | Мониторинг, оркестрация, контроль уязвимостей |
    | **SAST** | Статический анализ кода |
    | **DAST** | Динамический анализ приложения |
    | **Факторий** | Внутреннее хранилище зависимостей и артефакторий |
    | **OSA/SCA** | Композиционный анализ |
